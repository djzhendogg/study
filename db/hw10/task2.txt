2.0. Общий план
Запросу FreeSeats потребуется FlightId с фронта, затем он вернет setof varchar(4), кастуем к нужному формату для отображения.
Сценарий покупки: BuyFree - вернет boolean:
 * true - отображаем успешное выполение операции.
 * false - пробрасываем алерт для обновления страницы и возвращаем на страницу с выбором мест.
 Сценарий бронирования: 
 1. требуем авторизацию CorrectPasword - вернет boolean:
	* true - отображаем успешное выполение операции, делается вход в личный кабинет, подгружаются необходимые данные.
	* false - выдаем алерт с просьбой поправить UserId и Pass.
 или регистрацию RegisterUser - вернет boolean:
	* true - отображаем успешное выполение операции, записываем пользователя в базу, иделается вход в личный кабинет, подгружаются необходимые данные.
	* false - выдаем алерт с просьбой придумать новый UserId.
 2. бронируем выбранное место Reserve - вернет boolean:
	* true - отображаем успешное выполение операции.
	* false - пробрасываем алерт для обновления страницы и возвращаем на страницу с выбором мест.
Подразумевается, что повторная авторизация не потребуется.
 

start transaction read write isolation level read committed;
select FreeSeats(:FlightId);
commit;

Через веб-приложение. На главной странице можно будет выбрать номер рейса из всех возможных 
(в идеале соберать информацию о городах вылета, прилета и дате, но для этого нужны доп. таблицы). 
Далее его перекинет на страницу со свободными местами, выданых с помощью процедуры FreeSeats. После выбора места ему предложит опцию бронирования или покупки.
В случае покупки, пользователя перекинет на страницу оформления, где авторизация будет опциональна (BuyFree).
В случае бронирования, сначала перекинет на страницу авторизации(CorrectPasword)/регистрации(RegisterUser). Далее на страницу с оформлением (Reserve).
В случае успешного бронирования/покупки придет уведомление, например на почту. Для брони можно будет посмотреть актуальность на сайте (в личном кабинете).
В случае безуспешной операции, придет просьба о перезагрузке страницы и выкинет на страницу с выбором мест и по новой.

start transaction read write isolation level read committed;
select CorrectPasword(:UserId, :Pass);
commit;

start transaction read write isolation level read committed;
select RegisterUser(:UserId, :Pass);
commit;

start transaction read write isolation level snapshot;
select Reserve(:UserId, :Pass, :FlightId:, :SeatNo);
commit;

start transaction read write isolation level snapshot;
select BuyFree(:FlightId:, :SeatNo);
commit;
